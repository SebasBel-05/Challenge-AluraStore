Análisis de Rendimiento de Tiendas
Este proyecto se centra en el análisis de datos de ventas de cuatro tiendas diferentes con el objetivo de evaluar su rendimiento y determinar cuál sería la opción más adecuada para ser vendida.

Fuentes de Datos
Los datos provienen de cuatro archivos CSV, cada uno representando las ventas de una tienda:

tienda_1.csv
tienda_2.csv
tienda_3.csv
tienda_4.csv
Metodología de Análisis
Se realizaron varios análisis clave para evaluar el desempeño de cada tienda:

Análisis de Facturación: Cálculo de los ingresos totales generados por cada tienda.
Ventas por Categoría: Distribución de las ventas por categorías de productos para identificar patrones y diversificación de inventario.
Calificación Promedio de Clientes: Evaluación de la satisfacción del cliente a través de la calificación promedio de los productos.
Productos Más y Menos Vendidos: Identificación de los productos con mayor y menor volumen de ventas.
Costo de Envío Promedio: Análisis del costo promedio asociado al envío de productos por tienda.
Conclusiones Clave
Tras un análisis exhaustivo, se llegó a las siguientes conclusiones:

Tienda 3: Se posiciona como la opción más sólida, destacándose por una alta calificación de clientes y una buena diversificación de productos, a pesar de ingresos ligeramente menores.
Tienda 2: Muestra una concentración de ventas más alta en un número limitado de productos.
Tienda 1: Genera los ingresos más altos, pero sufre de una calificación de cliente baja y costos de envío elevados.
Tienda 4: Es la tienda menos eficiente según los criterios evaluados. A pesar de tener el costo de envío más bajo, sus ingresos son los menores, la satisfacción del cliente no es sobresaliente y su alta concentración de ventas indica una dependencia de pocos productos.
Recomendación: Se concluye que el Sr. Juan debería considerar vender la Tienda 4 para financiar su nuevo emprendimiento, dada su ineficiencia global en comparación con las otras tiendas.
